# VM汉化组官方网站 v4

立即访问：<https://v4.vmct-cn.top/>

## 📖 使用与部署

我们推荐使用VSCode编辑器进行开发。

在开发前，请先安装[NodeJS](https://nodejs.org/zh-cn/download/prebuilt-installer)，推荐安装 Node22。

安装依赖：

```sh
npm install
```

开发预览：

```sh
npm run dev
```

构建：

```sh
npm run build
```
